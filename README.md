# 🚀 Leadnex | Instant AI Insights Platform

Leadnex is a modern, elegant, and fully interactive SaaS landing page and dashboard designed for AI-driven marketing analytics. This version features a **Single Page Application (SPA)** architecture using Vanilla JavaScript, providing a seamless transition between pages without refreshing the browser.

## ✨ Key Features

-   **Elegant Glassmorphism UI**: Uses high-end blur effects, radial gradients, and soft shadows for a premium aesthetic.
-   **Seamless SPA Navigation**: Experience instantaneous page switching (Home, Employees, Projects, Dashboard, etc.) via custom JavaScript routing.
-   **Interactive Dashboard**: A functional preview of the Leadnex engine including KPI metrics, campaign tracking, and AI assistant banners.
-   **Ultra-Responsive Design**: Tailored layouts for Mobile, Tablet, and Desktop using Tailwind's mobile-first grid system.
-   **Premium Typography**: Integrated with *Plus Jakarta Sans* for a modern, high-tech corporate feel.
-   **Micro-Interactions**: Hover scales, smooth transitions, and active state animations to enhance user engagement.

## 🛠️ Technology Stack

-   **HTML5**: Semantic structure.
-   **Tailwind CSS**: Advanced utility classes for modern styling and layout.
-   **Vanilla JavaScript**: Custom routing logic and UI state management.
-   **Lucide Icons**: Clean and consistent vector iconography.
-   **Google Fonts**: High-quality typography (Plus Jakarta Sans).
